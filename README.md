# Voorschoten Online Starter

Dit starter project bevat:

- Next.js App Router
- MySQL-koppeling via `mysql2`
- homepage met laatste artikelen
- artikel-detailpagina
- agenda-overzicht
- health endpoint

## Installatie

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Nodige databasegegevens

Zet in `.env.local`:

- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USER`
- `DB_PASSWORD`

## Tabellen die gebruikt worden

- `published_articles`
- `article_sources`
- `media_assets`
- `events`
